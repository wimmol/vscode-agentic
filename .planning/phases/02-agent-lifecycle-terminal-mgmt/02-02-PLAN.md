    Create `src/services/agent.service.ts`:

    ```typescript
    import type * as vscode from "vscode";
    import type { AgentEntry } from "../models/agent.js";
    import { AGENT_REGISTRY_KEY } from "../models/agent.js";
    import type { TerminalService } from "./terminal.service.js";
    import type { WorktreeService } from "./worktree.service.js";
    ```

    Implement `AgentService` class with constructor injection:
    - Constructor takes `state: vscode.Memento`, `worktreeService: WorktreeService`, `terminalService: TerminalService`
    - Private methods `getRegistry(): AgentEntry[]` reads from `state.get(AGENT_REGISTRY_KEY, [])` and `saveRegistry(entries: AgentEntry[])` writes via `state.update(AGENT_REGISTRY_KEY, entries)`

    Methods:
    - `async createAgent(repoPath: string, agentName: string, initialPrompt?: string): Promise<AgentEntry>`:
      1. Call `this.worktreeService.addWorktree(repoPath, agentName)` to create branch+worktree
      2. Create `AgentEntry` with status "created", createdAt = new Date().toISOString()
      3. Append to registry and save
      4. Return the entry

    - `getAgent(repoPath: string, agentName: string): AgentEntry | undefined`: Find in registry by compound key

    - `getAll(): AgentEntry[]`: Return full registry

    - `getForRepo(repoPath: string): AgentEntry[]`: Filter registry by repoPath

    - `async deleteAgent(repoPath: string, agentName: string): Promise<void>`:
      1. If agent not found, return silently
      2. Call `this.terminalService.disposeTerminal(repoPath, agentName)`
      3. Call `this.worktreeService.removeWorktree(repoPath, agentName)`
      4. Remove from registry and save

    - `async focusAgent(repoPath: string, agentName: string): Promise<void>`:
      1. Get agent entry. If not found, return
      2. If status is "created", "finished", or "error": look up worktree path from `this.worktreeService.getManifest(repoPath).find(w => w.agentName === agentName)`. If no worktree found, return. Create terminal via `this.terminalService.createTerminal(repoPath, agentName, worktreeEntry.path, agent.initialPrompt)`, call `.show()`, update status to "running"
      3. If status is "running": call `this.terminalService.showTerminal(repoPath, agentName)`

    - `async updateStatus(repoPath: string, agentName: string, status: AgentStatus, exitCode?: number): Promise<void>`:
      Find agent in registry, update status and exitCode, save

    - `async reconcileOnActivation(): Promise<void>`:
      Get all entries. For any with status "running", set status to "created" and clear exitCode. Save if any changed.

    Write comprehensive tests in `test/unit/agent.service.test.ts`:
    - Mock WorktreeService and TerminalService as vi.fn() objects
    - Use createMockMemento() from vscode mock for state
    - Test all behaviors listed above
    - Verify correct interaction order (e.g., disposeTerminal before removeWorktree in deleteAgent)
