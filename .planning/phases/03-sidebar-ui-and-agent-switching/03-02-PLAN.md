         - Public `getActiveAgent()` returns the active agent tuple or undefined
         - Public async `switchToAgent(repoPath: string, agentName: string): Promise<void>`:
           a. Always call `agentService.focusAgent(repoPath, agentName)` -- handles terminal show/create
           b. Get worktree path from `worktreeService.getManifest(repoPath).find(w => w.agentName === agentName)?.path`
           c. If worktreePath found:
              - Replace ALL workspace folders with only the agent's worktree:
                ```typescript
                const folders = vscode.workspace.workspaceFolders ?? [];
                const worktreeUri = vscode.Uri.file(worktreePath);
                // Skip if already the sole workspace folder
                const alreadySole = folders.length === 1 && folders[0].uri.fsPath === worktreeUri.fsPath;
                if (!alreadySole) {
                  vscode.workspace.updateWorkspaceFolders(0, folders.length, { uri: worktreeUri, name: agentName });
                }
                ```
              - Focus file explorer: `await vscode.commands.executeCommand("workbench.view.explorer")`
              - Open a file from the worktree in the editor (best-effort):
                ```typescript
                const readmePath = vscode.Uri.joinPath(worktreeUri, "README.md");
                try {
                  const doc = await vscode.workspace.openTextDocument(readmePath);
                  await vscode.window.showTextDocument(doc, { preview: true, preserveFocus: true });
                } catch {
                  // README.md doesn't exist -- acceptable fallback.
                }
                ```
           d. Update `this.activeAgent = { repoPath, agentName }`
